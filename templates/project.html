Como Criar um bom README

Nesse projeto eu quis mostrar a minha visão de como um bom README pode ajudar a mostrar seu projeto.

Com isso criei um projeto ensinando ao meu ver como criar um README utilizando as linguagens de marcação deforma a você mesmo criar e estilizar com o Markdown.

O Markdown assim como o Html é uma linguagem de marcação que o Github utiliza para montar o seu projeto de Readme.

E como acredito que contribuir com a comunidade é a melhor forma de interagir e mostrar meu trabalho criei esse projeto, para de alguma forma apresentar a novas pessoas como um Github estrategico faz a diferença na nossa jornada.
